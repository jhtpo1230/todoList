# 🗂️ TodoList

> Express + MySQL 백엔드, React 프론트엔드로 구성된 풀스택 프로젝트로, <br>
> 로그인 · 회원가입 기능이 포함된 **개인/팀용 To-Do 관리 웹페이지**

---
## 💡 주요 기능

-  **할 일 등록 / 수정 / 삭제 / 완료 표시**
-  **로그인 / 회원가입**
-  **팀 구성 및 팀원 초대 / 팀별 To-Do 관리 페이지**

---

## 💻 기술 스택

| 구분 | 사용 기술 |
|------|------------|
| **Frontend** | React + Axios |
| **Backend** | Node.js + Express + mysql2 |
| **DB** | MariaDB (Docker 컨테이너) |
| **환경 변수 관리** | dotenv |
| **형상 관리** | Git / GitHub |

---

## ⚙️ 실행 방법

### 1️⃣ BackEnd 실행
```bash
cd backend
npm install
node server.js
```
### 2️⃣ FrontEnd 실행
```bash
cd frontend
npm install
npm start
```
